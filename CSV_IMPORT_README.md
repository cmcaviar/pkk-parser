# Импорт кадастровых номеров из CSV файла

## Описание

В GUI приложение добавлена функция импорта кадастровых номеров из CSV файлов. Это позволяет быстро загрузить большое количество номеров без ручного ввода.

## Как использовать

1. **Запустите приложение**
   ```bash
   python gui_app.py
   ```

2. **Нажмите кнопку "📄 Импорт CSV"**

3. **Выберите CSV файл** с кадастровыми номерами

4. **Номера автоматически загрузятся** в текстовое поле

5. **Нажмите "📥 Загрузить данные"** для обработки

## Поддерживаемые форматы CSV

### 1. Простой список (один номер на строку)

```
77:05:0001016:22
77:06:0002008:40
77:01:0005016:3275
```

### 2. CSV с заголовками (разделитель: запятая)

```csv
cadastral_number,address,area
77:05:0001016:22,Москва ул. Ленина д.1,1500
77:06:0002008:40,Москва ул. Пушкина д.2,2000
77:01:0005016:3275,Москва ул. Гоголя д.3,1800
```

### 3. CSV с заголовками (разделитель: точка с запятой)

```csv
кадастровый_номер;адрес;площадь
77:05:0001016:22;Москва ул. Ленина д.1;1500
77:06:0002008:40;Москва ул. Пушкина д.2;2000
```

### 4. Несколько номеров в одной строке

```
77:05:0001016:22, 77:06:0002008:40, 77:01:0005016:3275
```

## Поддерживаемые названия колонок

Парсер автоматически распознаёт следующие названия колонок:
- `cadastral_number`
- `кадастровый_номер`
- `кадастровый номер`
- `cadastral`
- `кадастр`
- `номер`
- `number`
- `cn`

Если названия не совпадают, будет использована **первая колонка**.

## Поддерживаемые кодировки

- UTF-8
- Windows-1251 (CP1251)

Кодировка определяется автоматически.

## Особенности

✅ **Автоматическое удаление дубликатов** - одинаковые номера будут удалены
✅ **Валидация номеров** - проверка формата XX:XX:XXXXXXX:XX
✅ **Пропуск комментариев** - строки начинающиеся с `#` игнорируются
✅ **Добавление к существующим** - если в поле уже есть номера, новые добавятся
✅ **Обработка ошибок** - понятные сообщения об ошибках при неверном формате

## Примеры использования

### Пример 1: Импорт простого списка

1. Создайте файл `my_cadastral.txt`:
   ```
   77:05:0001016:22
   77:06:0002008:40
   ```

2. Нажмите "📄 Импорт CSV"

3. Выберите файл `my_cadastral.txt`

4. Номера загрузятся автоматически

### Пример 2: Импорт из Excel (сохранённого как CSV)

1. В Excel создайте таблицу:
   | cadastral_number | address |
   |------------------|---------|
   | 77:05:0001016:22 | Москва  |
   | 77:06:0002008:40 | Москва  |

2. Сохраните как CSV (File → Save As → CSV)

3. Импортируйте через кнопку "📄 Импорт CSV"

## Исправление проблем

### Ошибка "В файле не найдено кадастровых номеров"

**Причина**: Формат номеров не распознан

**Решение**:
- Проверьте формат номеров (должно быть XX:XX:XXXXXXX:XX)
- Убедитесь что номера состоят только из цифр и двоеточий
- Проверьте что файл не пустой

### Ошибка "Не удалось определить кодировку файла"

**Причина**: Файл имеет неподдерживаемую кодировку

**Решение**:
- Пересохраните файл в UTF-8
- Используйте Notepad++ или другой редактор для смены кодировки
