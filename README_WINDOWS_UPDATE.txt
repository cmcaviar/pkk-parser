БЫСТРАЯ ИНСТРУКЦИЯ:

1. Скопируйте ВСЕ файлы проекта на Windows

2. Откройте командную строку (cmd) в папке проекта

3. Соберите новый EXE:

   python build_exe.py

   (выберите вариант 1)

4. Запустите новый EXE: dist/nspd_parser.exe

5. Включите "Dev логи" в интерфейсе

6. Протестируйте с номером: 77:01:0001003:1926
