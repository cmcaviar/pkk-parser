## 📊 Структура Excel результата

### 21 колонка (A-U):

**A:** Кадастровый номер участка

**B-H:** Данные земельного участка (дублируются для каждого объекта)
- B: Вид объекта недвижимости
- C: Вид земельного участка
- D: Адрес
- E: Площадь декларированная
- F: Вид разрешенного использования
- G: Форма собственности
- H: Кадастровая стоимость

**I-U:** Данные объектов на участке (уникальные для каждого объекта)
- I: Вид объекта недвижимости
- J: Кадастровый номер объекта
- K: Назначение
- L: Площадь общая
- M: Форма собственности
- N: Кадастровая стоимость
- O: Удельный показатель кадастровой стоимости
- P: Количество этажей (в том числе подземных)
- Q: Количество подземных этажей
- R: Материал стен
- S: Завершение строительства
- T: Ввод в эксплуатацию
- U: Сведения о культурном наследии

### Логика заполнения:
- **1 участок + 0 объектов** → 1 строка (A-H заполнены, I-U пустые)
- **1 участок + 3 объекта** → 3 строки (A-H дублируются, I-U уникальны)

## 🔧 Технические детали

### API
- **База:** requests (без Selenium)
- **Эндпоинт:** `https://nspd.gov.ru/api/data/...`
- **SSL:** Отключена верификация (self-signed cert)
- **Rate limiting:** ~5-10 сек на запрос
- **Обработка ошибок:** Автоматические повторы

### Excel
- **Библиотека:** openpyxl (создание) + xlwings (интеграция)
- **Формат:** .xlsx
- **Шаблон:** Строгий, 21 колонка
- **Форматирование:** Автоматическое (заголовки, цвета, границы)

### GUI (для .exe)
- **Фреймворк:** Tkinter (встроен в Python)
- **Многопоточность:** threading для UI отзывчивости
- **Логирование:** В реальном времени в GUI
- **Прогресс:** Прогресс-бар + текстовый статус

### Сборка .exe
- **Инструмент:** PyInstaller 5.13+
- **Режим:** --onefile (один файл)
- **Тип:** --windowed (без консоли)
- **Размер:** ~50-70 МБ (Python runtime + все библиотеки)
- **Совместимость:** Windows 7+

## 🐛 Известные ограничения

### API
- ⚠️ Скорость: ~5-10 секунд на запрос (ограничение API)
- ⚠️ Иногда API недоступен (проблемы на стороне НСПД)
- ⚠️ Не все поля всегда заполнены (зависит от данных в базе)

### .exe приложение
- ⚠️ Медленный первый запуск (~5-10 сек)
- ⚠️ Большой размер файла (~50-70 МБ)
- ⚠️ Антивирусы могут ругаться (false positive)
- ⚠️ Только Windows

### Excel интеграция
- ⚠️ Требует настройки xlwings
- ⚠️ На Mac: проблемы с autofit колонок (обойдены)
- ⚠️ Требует разрешения макросов

## 📞 Поддержка

### Если возникли проблемы:

1. **Найдите нужную документацию** (см. таблицы выше)
2. **Прочитайте раздел "Решение проблем"** в соответствующей инструкции
3. **Проверьте версию Python** (должна быть 3.8+)
4. **Проверьте зависимости:** `pip install -r requirements.txt`

### Частые проблемы:

**"ModuleNotFoundError"** → `pip install -r requirements.txt`

**"Участок не найден"** → Проверьте кадастровый номер на https://pkk.rosreestr.ru

**"xlwings не работает"** → `xlwings addin install` + перезапустите Excel

**"Антивирус блокирует .exe"** → Добавьте в исключения (это false positive)
