# 📘 Руководство пользователя - НСПД Парсер

Простая инструкция для работы с программой получения кадастровых данных.

---

## 🎯 Что делает эта программа?

**НСПД Парсер** автоматически получает подробные данные о земельных участках и объектах недвижимости по кадастровым номерам из Национальной системы пространственных данных (НСПД).

**Входные данные:** Кадастровые номера (например: `77:05:0001016:22`)

**Результат:** Excel файл с подробной информацией:
- Данные земельного участка (адрес, площадь, стоимость, ВРИ и т.д.)
- Список всех объектов на участке (здания, сооружения, помещения)
- Детальные характеристики каждого объекта

## 🚀 Быстрый старт

### Шаг 1: Запуск программы

1. Найдите файл `nspd_parser.exe`
2. Двойной клик по файлу
3. Откроется окно программы

**Примечание:** Первый запуск может занять 5-10 секунд (программа распаковывает компоненты).

### Шаг 2: Ввод кадастровых номеров

В верхней части окна есть большое текстовое поле.

**Введите кадастровые номера** - по одному на каждую строку:

```
77:05:0001016:22
77:06:0002008:40
77:01:0005016:3275
```

**Правила ввода:**
- ✅ Один номер на строку
- ✅ Можно вводить много номеров (хоть 100)
- ✅ Можно копировать из Excel / Word / Блокнота
- ✅ Пустые строки игнорируются
- ✅ Строки начинающиеся с `#` - комментарии (игнорируются)

**Пример с комментариями:**
```
# Участки в Москве
77:05:0001016:22
77:06:0002008:40

# Участок в Подмосковье
50:23:0030214:156
```

### Шаг 3: Запуск обработки

1. Нажмите кнопку **"📥 Загрузить данные"**
2. Выберите **куда сохранить** Excel файл
   - Программа предложит имя файла автоматически
   - Можете изменить имя и место сохранения
3. Нажмите **"Сохранить"**

### Шаг 4: Ожидание результата

**Во время работы программы:**

1. **Прогресс-бар** показывает процент выполнения
   ```
   ▓▓▓▓▓▓▓░░░░░░░ 40% (2/5)
   ```

2. **Лог выполнения** показывает что происходит:
   ```
   ✅ Найдено номеров: 3
   ⏳ [1/3] 77:05:0001016:22
   ✅ Участок: г. Москва, Красная площадь...
   ✅ Объектов найдено: 2
   ✅ Данные обработаны
   ```

3. **Не закрывайте программу!** Дождитесь завершения.

**Скорость работы:**
- ~5-10 секунд на один кадастровый номер
- Если участков 10 → примерно 1-2 минуты
- Если участков 100 → примерно 10-15 минут

### Шаг 5: Результат

По завершении:

1. Появится окно: **"Обработка завершена!"**
   - Показывает сколько номеров обработано
   - Сколько успешно, сколько с ошибками
   - Путь к созданному файлу

2. Откройте созданный **Excel файл**
   - Программа НЕ открывает его автоматически
   - Найдите файл по указанному пути
   - Откройте в Excel / LibreOffice / Google Sheets

## 📊 Структура Excel файла

### Колонки (A-U, всего 21 колонка):

**A: Кадастровый номер**

**B-H: Сведения о земельном участке**
- B: Вид объекта недвижимости
- C: Вид земельного участка
- D: Адрес
- E: Площадь декларированная
- F: Вид разрешенного использования
- G: Форма собственности
- H: Кадастровая стоимость

**I-U: Сведения об объектах на участке**
- I: Вид объекта недвижимости
- J: Кадастровый номер объекта
- K: Назначение
- L: Площадь общая
- M: Форма собственности
- N: Кадастровая стоимость
- O: Удельный показатель кадастровой стоимости
- P: Количество этажей
- Q: Количество подземных этажей
- R: Материал стен
- S: Завершение строительства
- T: Ввод в эксплуатацию
- U: Сведения о культурном наследии

### Как читать результат:

**Если на участке 1 объект** → 1 строка в Excel

**Если на участке 3 объекта** → 3 строки в Excel
- Данные участка (B-H) дублируются во всех 3 строках
- Данные объектов (I-U) разные в каждой строке

**Если объектов нет** → 1 строка (только данные участка, колонки I-U пустые)

## ❓ Часто задаваемые вопросы

### В: Нужно ли устанавливать программу?

**О:** Нет! Просто запустите .exe файл. Никакой установки не требуется.

### В: Можно ли запустить на компьютере без Excel?

**О:** Да! Программа создаёт Excel файлы без установленного Excel. Открыть результат можно в:
- Microsoft Excel (если есть)
- LibreOffice Calc (бесплатно)
- Google Sheets (онлайн)
- Любой программе читающей .xlsx файлы

### В: Почему программа долго запускается первый раз?

**О:** Это нормально. При первом запуске программа распаковывает компоненты. Последующие запуски будут быстрее.

### В: Сколько номеров можно обработать за раз?

**О:** Теоретически - неограниченно. Практически:
- До 50 номеров - комфортно (5-10 минут)
- До 200 номеров - терпимо (30-40 минут)
- Больше 200 - лучше разбить на части

### В: Можно ли остановить обработку?

**О:** Нет встроенной кнопки "Стоп". Можно закрыть программу крестиком, но тогда результат не сохранится.

### В: Что делать если номер не найден?

**О:** В логе появится:
```
❌ Ошибка: Участок не найден
```
Программа продолжит обработку остальных номеров. В Excel этот номер будет отмечен как ошибка.

### В: Программа зависла, что делать?

**О:** Если прогресс-бар не меняется больше 2 минут:
1. Проверьте подключение к интернету
2. Проверьте что API НСПД доступен
3. Попробуйте закрыть и запустить заново
4. Проверьте правильность кадастрового номера

### В: Безопасна ли программа?

**О:** Да! Программа:
- ✅ Не собирает никаких данных
- ✅ Не отправляет данные никуда, кроме API НСПД
- ✅ Не требует прав администратора
- ✅ Не изменяет системные файлы
- ✅ Исходный код открыт (можно проверить)

**Антивирус может ругаться** - это ложное срабатывание. Упакованные .exe файлы иногда вызывают подозрение. Добавьте программу в исключения.

### В: Программа не запускается

**Возможные причины:**

1. **Windows Defender блокирует**
   - Нажмите "Подробнее" → "Выполнить в любом случае"

2. **Антивирус блокирует**
   - Добавьте nspd_parser.exe в исключения

3. **Старая версия Windows**
   - Требуется Windows 7 или новее

4. **Повреждён файл**
   - Скачайте заново

### В: Где хранятся временные файлы?

**О:** Программа создаёт временные файлы в:
```
C:\Users\[ваше_имя]\AppData\Local\Temp\_MEI*
```

Эти файлы автоматически удаляются при закрытии программы.

## 🐛 Решение проблем

### Ошибка: "Не удаётся подключиться к API"

**Причины:**
- Нет интернета
- Сайт НСПД недоступен
- Блокировка файрволом

**Решение:**
1. Проверьте интернет-соединение
2. Попробуйте открыть https://nspd.gov.ru в браузере
3. Проверьте настройки файрвола

### Ошибка: "Участок не найден"

**Причины:**
- Неправильный кадастровый номер
- Участок не внесён в систему
- Ошибка в базе НСПД

**Решение:**
1. Проверьте правильность номера
2. Проверьте формат: `XX:XX:XXXXXXX:XXXX`
3. Попробуйте найти участок на https://pkk.rosreestr.ru

### Ошибка: "Не удаётся сохранить файл"

**Причины:**
- Нет прав на запись в папку
- Файл с таким именем уже открыт в Excel
- Диск переполнен

**Решение:**
1. Выберите другую папку для сохранения
2. Закройте Excel если открыт файл с таким именем
3. Проверьте свободное место на диске

### Программа работает очень медленно

**Причины:**
- Медленный интернет
- Перегрузка API НСПД
- Много номеров для обработки

**Решение:**
- Это нормально, API работает медленно
- ~5-10 секунд на номер - это нормальная скорость
- Если очень долго - разбейте на меньшие партии

## 💡 Советы по использованию

### Подготовка списка номеров

**В Excel:**
1. Скопируйте колонку с номерами (Ctrl+C)
2. Вставьте в программу (Ctrl+V)

**Из текстового файла:**
1. Откройте .txt файл с номерами (по одному на строку)
2. Выделите всё (Ctrl+A)
3. Скопируйте (Ctrl+C)
4. Вставьте в программу (Ctrl+V)

### Обработка больших списков

Если номеров очень много (100+):

1. Разбейте на части по 50 номеров
2. Обработайте каждую часть отдельно
3. Объедините Excel файлы вручную

**Или:** Запустите несколько копий программы параллельно (по 50 номеров в каждой).

### Автоматизация

Если нужно обрабатывать данные регулярно:

1. Создайте текстовый файл `numbers.txt` со списком номеров
2. Копируйте из него в программу
3. Обновляйте файл по мере необходимости
