# ⚡ Быстрый старт - Windows .exe приложение

Создание standalone .exe файла для Windows в 3 шага.

---

## 📋 Что вам нужно

- ✅ Компьютер с **Windows**
- ✅ **Python 3.8+** установлен
- ✅ Исходники проекта `pkk-parser/`

---

## 🚀 3 шага до .exe

### Шаг 1: Установка зависимостей

Откройте **Command Prompt** в папке проекта:

```cmd
pip install -r requirements.txt
pip install pyinstaller
```

### Шаг 2: Сборка .exe

```cmd
python build_exe.py
```

Выберите **"1"** (Собрать .exe файл)

Подождите 2-3 минуты...

### Шаг 3: Готово!

Готовый файл: **`dist/nspd_parser.exe`**

✅ Размер: ~50-70 МБ
✅ Полностью автономный
✅ Не требует Python
✅ Работает на любом Windows 7+

---

## 🎮 Использование

1. Двойной клик на `nspd_parser.exe`
2. Введите кадастровые номера (по одному на строку)
3. Нажмите "Загрузить данные"
4. Выберите где сохранить Excel файл
5. Ждите завершения
6. Откройте созданный Excel файл
