# Контекстное меню (правая кнопка мыши)

## Описание

В GUI приложение добавлено полноценное контекстное меню, которое появляется при нажатии правой кнопки мыши на текстовом поле с кадастровыми номерами.

## Способы вставки текста

Теперь поддерживаются **3 способа** вставки текста:

### 1. Клавиатурные сочетания
- **Windows/Linux:** `Ctrl + V`
- **macOS:** `Cmd + V`

### 2. Контекстное меню (правая кнопка мыши)
- Щёлкните **правой кнопкой мыши** по текстовому полю
- Выберите **"Вставить"**

### 3. Виртуальное событие вставки
- Автоматически перехватывает все виды вставки

## Функции контекстного меню

При клике правой кнопкой мыши доступны следующие опции:

| Команда | Действие | Сочетание клавиш |
|---------|----------|------------------|
| **Вырезать** | Вырезает выделенный текст в буфер обмена | `Ctrl+X` |
| **Копировать** | Копирует выделенный текст в буфер обмена | `Ctrl+C` |
| **Вставить** | Вставляет текст из буфера обмена | `Ctrl+V` |
| ── | ── | ── |
| **Выделить всё** | Выделяет весь текст в поле | `Ctrl+A` |
| ── | ── | ── |
| **Очистить** | Очищает текстовое поле | - |

## Особенности

### ✅ Автоматическая обработка Placeholder

При вставке через контекстное меню:
1. Placeholder текст автоматически удаляется
2. Цвет текста меняется на чёрный
3. Вставляется содержимое буфера обмена

### ✅ Кросс-платформенность

Контекстное меню работает на всех платформах:
- **Windows:** правая кнопка мыши (`<Button-3>`)
- **Linux:** правая кнопка мыши (`<Button-3>`)
- **macOS:** правая кнопка мыши / Control+Click (`<Button-2>`)

### ✅ Умная активация команд

- **Вырезать/Копировать/Выделить всё:** работают только если placeholder неактивен
- **Вставить:** автоматически удаляет placeholder перед вставкой
- **Очистить:** всегда доступна

## Примеры использования

### Пример 1: Вставка через контекстное меню

1. Скопируйте кадастровый номер (например: `77:05:0001016:22`)
2. Откройте приложение
3. **Щёлкните правой кнопкой мыши** по текстовому полю
4. Выберите **"Вставить"**
5. Номер вставится, placeholder исчезнет

### Пример 2: Копирование текста

1. Введите или вставьте кадастровые номера
2. Выделите нужный текст мышью
3. **Щёлкните правой кнопкой мыши**
4. Выберите **"Копировать"**
5. Текст скопирован в буфер обмена

### Пример 3: Быстрая очистка

1. В текстовом поле есть номера
2. **Щёлкните правой кнопкой мыши**
3. Выберите **"Очистить"**
4. Поле очистится, placeholder вернётся

    self.context_menu = tk.Menu(self.text_input, tearoff=0)

    self.context_menu.add_command(

### Показ контекстного меню (gui_app.py:225-230)

```python
def _show_context_menu(self, event):
    """Показать контекстное меню"""
    try:
        self.context_menu.tk_popup(event.x_root, event.y_root)
    finally:
        self.context_menu.grab_release()
```

### Обработка вставки (gui_app.py:248-259)

```python
def _context_paste(self):
    """Вставить текст (из контекстного меню)"""
    # Удаляем placeholder если активен
    if self.is_placeholder_active:
        self.text_input.delete("1.0", tk.END)
        self.text_input.config(fg="black")
        self.is_placeholder_active = False

    try:
        self.text_input.event_generate("<<Paste>>")
    except tk.TclError:
        pass
```

### Привязка событий (gui_app.py:101-102)

```python
self.text_input.bind("<Button-3>", self._show_context_menu)  # Windows/Linux
self.text_input.bind("<Button-2>", self._show_context_menu)  # macOS
```
