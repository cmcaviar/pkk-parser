# 📊 Использование парсера НСПД из Excel

Подробная инструкция по запуску парсера прямо из Excel без командной строки.

---

## 🎯 Что это дает

- ✅ Вписываете кадастровые номера в колонку A
- ✅ Нажимаете кнопку "Загрузить данные"
- ✅ Результаты появляются в том же листе (колонки A-U)
- ✅ Никаких консолей и командной строки!

---

## 📋 Требования

1. **Windows** с установленным Microsoft Excel
2. **Python 3.8+** (должен быть установлен)
3. **Библиотеки Python** (устанавливаются один раз)

---

## ⚙️ Установка (делается один раз)

### Шаг 1: Установите Python

Если Python еще не установлен:

1. Скачайте с https://www.python.org/downloads/
2. При установке **ОБЯЗАТЕЛЬНО** поставьте галочку **"Add Python to PATH"**
3. Нажмите "Install Now"

**Проверка:**
```cmd
python --version
```

### Шаг 2: Установите зависимости Python

Откройте командную строку (только один раз!):

```cmd
cd C:\путь\к\pkk-parser
pip install -r requirements.txt
```

Будут установлены:
- `requests` - для работы с API
- `openpyxl` - для работы с Excel файлами
- `xlwings` - для интеграции Python с Excel

### Шаг 3: Установите xlwings add-in в Excel

В командной строке выполните:

```cmd
xlwings addin install
```

Эта команда добавит xlwings в Excel как надстройку.

**Важно:** После установки перезапустите Excel!

### Шаг 4: Настройте Excel для работы с макросами

1. Откройте Excel
2. Файл → Параметры → Центр управления безопасностью
3. Параметры центра управления безопасностью
4. Параметры макросов
5. Выберите: **"Включить все макросы"** (или "Отключить все макросы с уведомлением")
6. Поставьте галочку: **"Доверять доступ к объектной модели проектов VBA"**
7. Нажмите OK

## 📝 Настройка Excel файла

### Вариант А: Создать файл вручную

1. Создайте новый Excel файл (`.xlsm` - с поддержкой макросов)
2. Нажмите `Alt + F11` (откроется редактор VBA)
3. Insert → Module
4. Скопируйте код из файла `excel_macro.vba`
5. Сохраните и закройте редактор VBA
6. Сохраните файл как `.xlsm` (Excel Macro-Enabled Workbook)

### Вариант Б: Использовать готовый шаблон

Откройте файл `template_nspd.xlsm` (будет создан ниже)

## 🚀 Использование

### Подготовка данных

1. Откройте ваш Excel файл с макросами
2. В колонке **A** впишите кадастровые номера:

```
Строка 1: (здесь будут заголовки после выполнения)
Строка 2: 77:05:0001016:22
Строка 3: 77:06:0002008:40
Строка 4: ...
```

### Запуск обработки

**Способ 1: Через кнопку (если добавили)**

Нажмите кнопку "Загрузить данные НСПД" на листе

**Способ 2: Через меню**

1. Вкладка "Разработчик" → Макросы
2. Выберите `LoadNSPDData`
3. Нажмите "Выполнить"

**Способ 3: Горячая клавиша**

Назначьте сочетание клавиш (например, `Ctrl+Shift+L`)

### Что происходит

1. Python скрипт читает номера из колонки A
2. Для каждого номера:
   - Загружает данные участка
   - Загружает данные всех объектов на участке
   - Записывает результат в следующие колонки (B-U)
3. В строке состояния Excel показывается прогресс
4. По завершении появляется окно с результатами

### Результат

Данные появляются в том же листе:

- **Строка 1:** Заголовки колонок (A-U)
- **Строки 2+:** Данные
  - Если на участке 1 объект → 1 строка
  - Если на участке 5 объектов → 5 строк (данные участка дублируются)
  - Если объектов нет → 1 строка (колонки I-U с "-")

---

## 📊 Структура результата

### Колонки A-H: Данные участка

| Колонка | Название |
|---------|----------|
| A | Кадастровый номер объекта недвижимости |
| B | Вид объекта недвижимости |
| C | Вид земельного участка |
| D | Адрес |
| E | Площадь декларированная |
| F | Вид разрешенного использования |
| G | Форма собственности |
| H | Кадастровая стоимость |

### Колонки I-U: Данные объектов

| Колонка | Название |
|---------|----------|
| I | Вид объекта недвижимости |
| J | Кадастровый номер |
| K | Назначение |
| L | Площадь общая |
| M | Форма собственности |
| N | Кадастровая стоимость |
| O | Удельный показатель кадастровой стоимости |
| P | Количество этажей (в том числе подземных) |
| Q | Количество подземных этажей |
| R | Материал стен |
| S | Завершение строительства |
| T | Ввод в эксплуатацию |
| U | Сведения о культурном наследии |

## 🔧 Добавление кнопки на лист Excel

### Шаг 1: Показать вкладку "Разработчик"

1. Файл → Параметры → Настроить ленту
2. В правой колонке поставьте галочку **"Разработчик"**
3. OK

### Шаг 2: Создать кнопку

1. Вкладка "Разработчик" → Вставить → Кнопка (элемент управления формы)
2. Нарисуйте кнопку на листе
3. В появившемся окне выберите макрос `LoadNSPDData`
4. Нажмите OK
5. Правой кнопкой на кнопке → Изменить текст → "Загрузить данные НСПД"

Теперь можно нажимать кнопку для запуска!

## 🐛 Решение проблем

### Ошибка: "xlwings не настроен"

**Решение:**
```cmd
pip install xlwings
xlwings addin install
```

Перезапустите Excel.

### Ошибка: "Файл run_from_excel.py не найден"

**Причина:** Excel не может найти Python скрипт.

**Решение:** Убедитесь что:
1. Excel файл находится в той же папке что и `run_from_excel.py`
2. Или измените путь в VBA коде:
   ```vba
   scriptPath = "C:\полный\путь\к\run_from_excel.py"
   ```

### Ошибка: "Не удается выполнить макрос"

**Причина:** Макросы отключены.

**Решение:**
1. Файл → Параметры → Центр управления безопасностью
2. Включите макросы
3. Поставьте галочку "Доверять доступ к объектной модели проектов VBA"

### Ошибка: 403 Forbidden

**Причина:** Временная блокировка API.

**Решение:** Подождите несколько минут и попробуйте снова.

### Python не найден

**Причина:** Python не в PATH.

**Решение:**
1. Найдите где установлен Python (обычно `C:\Users\ВашеИмя\AppData\Local\Programs\Python\Python3XX`)
2. Добавьте в PATH или укажите полный путь в VBA:
   ```vba
   pythonPath = "C:\Python3XX\python.exe"
   ```

## 💡 Советы

### 1. Сохраняйте перед запуском

Перед запуском макроса сохраните Excel файл на случай ошибки.

### 2. Не закрывайте Excel во время обработки

Дождитесь завершения или нажмите ESC для прерывания.

### 3. Большие списки

Для обработки >50 номеров рекомендуется разбить на части.

### 4. Скорость

Обработка одного участка с объектами занимает ~5-15 секунд (зависит от количества объектов).

### 5. Ошибки в данных

Если для какого-то номера произошла ошибка:
- Номер останется в колонке A
- В колонке B появится "ОШИБКА: ..."
- Ячейка будет подсвечена красным
