# История изменений НСПД Парсера

## [v1.4.0] - 2026-04-05

### 🐛 Критические исправления

#### Исправлена работа на Windows (проблема с прокси)
- **Было:** На Windows программа не получала данные из-за ошибки прокси `127.0.0.1:2080`
- **Стало:**
  - Отключены системные прокси: `trust_env = False`
  - Явное отключение прокси: `proxies = {'http': None, 'https': None}`
- **Файлы:** `api_client.py:98-106`

#### Исправлено поле "Площадь застройки" для сооружений
- **Было:** Площадь застройки не отображалась (искалось несуществующее поле `params_construction_area`)
- **Стало:** Используются правильные поля `params_built_up_area` и `built_up_area`
- **Результат:** Площадь застройки теперь корректно извлекается для всех сооружений
- **Файлы:** `api_client.py:567-568, 532`

### ✨ Новые поля для объектов недвижимости

#### Добавлены поля для сооружений:
1. **Наименование** - название здания/сооружения
2. **Площадь застройки** - для сооружений
3. **Протяжённость** - для линейных сооружений

**Изменённые файлы:**
- `models.py` - добавлены поля `name`, `building_area`, `length`
- `api_client.py` - расширен парсинг сооружений
- `excel_export.py` - расширена таблица с 21 до 24 колонок

**Структура Excel:**
- Было: колонки I-U (21 колонка)
- Стало: колонки I-X (24 колонки)
  - K: Наименование ✨ НОВОЕ
  - N: Площадь застройки ✨ НОВОЕ
  - O: Протяжённость ✨ НОВОЕ

## Известные ограничения

### Режим разработчика
- Логи не удаляются автоматически (нужна ручная очистка)
- Dev режим немного замедляет GUI (~1-2%)

### Импорт CSV
- Максимальный размер CSV файла: не ограничен, но рекомендуется до 10 000 строк
- Поддерживаемые кодировки: UTF-8, Windows-1251

### Общие
- Требуется доступ к API НСПД для загрузки данных
- Windows: требуется Python 3.7+
