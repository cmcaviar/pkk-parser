│ СПОСОБ 1: КЛАВИАТУРА                                                    │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  Windows/Linux:  Ctrl + V                                               │
│  macOS:          Cmd + V                                                │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────┐
│ СПОСОБ 2: ПРАВАЯ КНОПКА МЫШИ (РЕКОМЕНДУЕТСЯ)                            │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  1. Скопируйте кадастровые номера                                       │
│  2. Щёлкните ПРАВОЙ кнопкой мыши по текстовому полю                     │
│  3. Выберите "Вставить"                                                 │
│                                                                         │
│  ✅ Работает на всех версиях Windows!                                   │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘

├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  1. Нажмите кнопку "📄 Импорт CSV"                                      │
│  2. Выберите CSV файл с номерами                                        │

═══════════════════════════════════════════════════════════════════════════
  КОНТЕКСТНОЕ МЕНЮ (правая кнопка мыши)
═══════════════════════════════════════════════════════════════════════════

Доступные команды:

  • Вырезать          (Ctrl+X)  - вырезать выделенный текст
  • Копировать        (Ctrl+C)  - скопировать выделенный текст
  • Вставить          (Ctrl+V)  - вставить из буфера обмена
  ─────────────────────────────────────────────────────────────────────
  • Выделить всё      (Ctrl+A)  - выделить весь текст
  ─────────────────────────────────────────────────────────────────────
  • Очистить                    - очистить текстовое поле


═══════════════════════════════════════════════════════════════════════════
  ПРИМЕРЫ ИСПОЛЬЗОВАНИЯ
═══════════════════════════════════════════════════════════════════════════

ПРИМЕР 1: Вставка одного номера
────────────────────────────────
1. Скопируйте: 77:05:0001016:22
2. Правая кнопка мыши → "Вставить"
3. Нажмите "📥 Старт"


ПРИМЕР 2: Вставка нескольких номеров
─────────────────────────────────────
1. Скопируйте список:
   77:05:0001016:22
   77:06:0002008:40
   77:01:0005016:3275

2. Правая кнопка мыши → "Вставить"
3. Нажмите "📥 Старт"


ПРИМЕР 3: Импорт из Excel
──────────────────────────
1. В Excel сохраните список как CSV:
   Файл → Сохранить как → CSV (разделители - запятые)

2. В приложении нажмите "📄 Импорт CSV"
3. Выберите сохранённый файл
4. Номера загрузятся автоматически
5. Нажмите "📥 Старт"

  УСТРАНЕНИЕ ПРОБЛЕМ
═══════════════════════════════════════════════════════════════════════════

❌ "Не вставляется через Ctrl+V"
   → Используйте ПРАВУЮ КНОПКУ МЫШИ → "Вставить"

❌ "Текст вставляется, но серый"
   → Это placeholder. Кликните в поле и начните вводить или вставлять

❌ "Импорт CSV не работает"
   → Убедитесь, что файл содержит номера в формате XX:XX:XXXXXXX:XX
   → Проверьте, что файл сохранён в кодировке UTF-8 или Windows-1251
