# 🎯 Способы использования НСПД Парсера

Выберите наиболее удобный для вас способ работы с парсером.

---

## 📊 Сравнение способов

| Способ | Сложность | Требует Python | Требует Excel | Платформа | UX |
|--------|-----------|----------------|---------------|-----------|-----|
| **1. Standalone .exe** | ⭐ Легко | ❌ Нет | ❌ Нет | Windows | 🏆 Лучший |
| **2. Интеграция с Excel** | ⭐⭐ Средне | ✅ Да | ✅ Да | Win/Mac | 🥈 Хороший |
| **3. Командная строка** | ⭐⭐⭐ Сложно | ✅ Да | ❌ Нет | Любая | 🥉 Для продвинутых |
| **4. Python API** | ⭐⭐⭐⭐ Очень сложно | ✅ Да | ❌ Нет | Любая | Для разработчиков |

## 1️⃣ Standalone .exe приложение (Windows)

### ✅ Преимущества
- Самый простой способ
- Не требует установки Python
- Не требует Excel
- Красивый графический интерфейс
- Один файл для распространения

### ❌ Недостатки
- Только для Windows
- Большой размер файла (~50-70 МБ)
- Медленный первый запуск (~5-10 сек)

### 📚 Документация
- **[Быстрый старт](QUICKSTART_EXE.md)** - 3 шага до .exe
- **[Сборка .exe](BUILD_WINDOWS.md)** - полная инструкция для разработчика
- **[Руководство пользователя](USER_GUIDE_EXE.md)** - как использовать готовый .exe

### 🚀 Как начать
```cmd
# На Windows:
pip install -r requirements.txt
pip install pyinstaller
python build_exe.py
# Готовый файл: dist/nspd_parser.exe
```

## 2️⃣ Интеграция с Excel (Windows/Mac)

### ✅ Преимущества
- Работа прямо в Excel
- Результаты в том же листе
- Можно использовать существующий список номеров
- Подходит для регулярного использования

### ❌ Недостатки
- Требует установки Python и библиотек
- Требует настройки xlwings и VBA макросов
- Более сложная начальная настройка

### 📚 Документация

**Windows:**
- **[Быстрый старт](QUICKSTART_EXCEL.md)** - 3 шага
- **[Полная инструкция](EXCEL_GUIDE.md)** - подробно

**macOS:**
- **[Быстрый старт](QUICKSTART_MAC.md)** - 3 способа
- **[Полная инструкция](EXCEL_MAC_GUIDE.md)** - подробно

### 🚀 Как начать

**Windows:**
```cmd
pip install -r requirements.txt
xlwings addin install
# Настройте макрос по инструкции EXCEL_GUIDE.md
```

**macOS:**
```bash
pip3 install -r requirements.txt
xlwings addin install
# Настройте макрос по инструкции EXCEL_MAC_GUIDE.md
```

## 3️⃣ Командная строка (Любая ОС)

### ✅ Преимущества
- Быстрый запуск
- Можно использовать в скриптах
- Не требует графического интерфейса
- Подходит для автоматизации

### ❌ Недостатки
- Требует базовых знаний командной строки
- Требует установки Python
- Менее удобный интерфейс

### 📚 Документация
- **[Windows инструкция](WINDOWS_GUIDE.md)** - для Windows
- **[README.md](README.md)** - общая документация

### 🚀 Как начать

```bash
# 1. Установите зависимости
pip install -r requirements.txt

# 2. Используйте готовые примеры

# Один номер
python example_single.py

# Несколько номеров
python example_batch.py

# Из текстового файла
python example_from_file.py

# Или напишите свой скрипт (см. README.md)
```

## 4️⃣ Python API (Для разработчиков)

### ✅ Преимущества
- Полный контроль
- Можно интегрировать в свои проекты
- Расширяемость
- Доступ ко всем функциям

### ❌ Недостатки
- Требует знания Python
- Самый сложный способ
- Нужно писать код

### 📚 Документация
- **[README.md](README.md)** - примеры использования API
- **[COMPLETE.md](COMPLETE.md)** - полная техническая документация
- **api_client.py** - исходный код API клиента

### 🚀 Как начать

```python
from api_client import NSPDAPIClient
from models import ParseResult
from excel_export import create_excel_with_template

# Создаём клиент
client = NSPDAPIClient()

try:
    # Получаем данные
    result = client.get_full_parcel_info_with_objects("77:05:0001016:22")

    # Формируем результат
    parse_result = ParseResult(
        cadastral_number="77:05:0001016:22",
        parcel=result['parcel_data'],
        objects=result['objects_data'],
        status="Успешно"
    )

    # Экспортируем в Excel
    create_excel_with_template([parse_result], "output.xlsx")

finally:
    client.close()
```
