## Интерактивные тесты

### 1. Тест вставки и контекстного меню (1 минута)

```bash
python test_paste.py
```

**Что сделать:**
1. Скопируйте: `77:05:0001016:22`
2. Вставьте через **правую кнопку мыши** → "Вставить"
3. Попробуйте "Копировать", "Выделить всё", "Очистить"

---

### 2. Режим разработчика (1 минута)

```bash
python test_dev_mode.py
```

**Что проверить:**
- [ ] Включите "🔧 Dev логи"
- [ ] В логах появляются [DEBUG] сообщения
- [ ] Кнопка "📄 Экспорт логов" работает
- [ ] Логи сохраняются в `logs/`

---

### 3. Основное приложение (3 минуты)

```bash
python gui_app.py
```

**Чек-лист:**
- [ ] Скопируйте несколько номеров
- [ ] Вставьте через **правую кнопку мыши**
- [ ] Нажмите "📄 Импорт CSV" → выберите `test_cadastral_simple.csv`
- [ ] Включите "🔧 Dev логи" и посмотрите детальные логи
- [ ] Нажмите "📥 Старт" (если есть доступ к API)
- [ ] Проверьте созданный Excel файл
