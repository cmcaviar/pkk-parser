## 📋 План тестирования

### ✅ Тест 1: Вставка и контекстное меню (2 минуты)

**Команда:**
```bash
python test_paste.py
```

**Что делать:**
1. Скопируйте текст: `77:05:0001016:22`
2. Кликните в текстовое поле
3. **Способ 1:** Нажмите `Ctrl+V` → текст должен вставиться
4. Очистите поле
5. **Способ 2:** Щёлкните ПРАВОЙ кнопкой мыши → "Вставить" → текст должен вставиться
6. Выделите текст мышью
7. Правая кнопка → "Копировать"
8. Правая кнопка → "Вставить"
9. Правая кнопка → "Выделить всё"
10. Правая кнопка → "Очистить"

**Ожидаемый результат:**
- ✅ Все операции работают
- ✅ Placeholder автоматически исчезает при вставке
- ✅ Контекстное меню открывается по правой кнопке

## 🔧 Устранение проблем

### Проблема: Тесты не запускаются

**Решение:**
```bash
# Проверьте Python версию (нужен 3.7+)
python --version

# Установите зависимости
pip install -r requirements.txt
```

### Проблема: GUI не открывается

**Решение:**
```bash
# Проверьте Tkinter
python -m tkinter

# Если ошибка, установите:
# Ubuntu/Debian:
sudo apt-get install python3-tk

# macOS (через Homebrew):
brew install python-tk
```

### Проблема: CSV импорт не работает

**Решение:**
- Проверьте, что CSV файлы находятся в той же папке
- Проверьте формат номеров (XX:XX:XXXXXXX:XX)
- Проверьте кодировку файла (UTF-8 или Windows-1251)
