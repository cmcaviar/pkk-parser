## 📋 Установка (5 минут)

### 1. Установите зависимости

```cmd
cd C:\путь\к\pkk-parser
pip install -r requirements.txt
xlwings addin install
```

### 2. Перезапустите Excel

---

## 🚀 Использование (3 шага)

### 1. Создайте Excel файл

1. Создайте новый файл Excel
2. Сохраните как `.xlsm` (с поддержкой макросов)
3. Сохраните в ту же папку где `run_from_excel.py`

### 2. Добавьте макрос

1. Нажмите `Alt + F11`
2. Insert → Module
3. Скопируйте код из `excel_macro.vba`
4. Сохраните (Ctrl + S)

### 3. Используйте

1. Впишите кадастровые номера в колонку A (с A2):
   ```
   A2: 77:05:0001016:22
   A3: 77:06:0002008:40
   ```

2. Нажмите `Alt + F8`
3. Выберите `LoadNSPDData`
4. Нажмите "Выполнить"

**Готово!** Данные появятся в колонках A-U.
