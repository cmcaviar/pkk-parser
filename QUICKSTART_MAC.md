# ⚡ Быстрый старт на macOS

3 способа протестировать работу с Excel на Mac.

---

## 🎯 Способ 1: Простой тест (рекомендуется)

Создаст новый Excel файл и обработает один номер:

```bash
# 1. Установите зависимости
pip3 install -r requirements.txt

# 2. Запустите тест
python3 test_excel_mac.py --simple
```

Excel откроется с результатами!

---

## 🎯 Способ 2: С вашим Excel файлом

Работает с уже открытым Excel:

```bash
# 1. Откройте Excel, впишите номера в колонку A (с A2)
# Пример:
# A2: 77:05:0001016:22
# A3: 77:06:0002008:40

# 2. Запустите в Terminal:
python3 test_excel_mac.py

# 3. Нажмите Enter
```

Данные появятся в том же файле!

---

## 🎯 Способ 3: Через VBA макрос

Для постоянного использования:

```bash
# 1. Установите xlwings add-in
xlwings addin install

# 2. Создайте .xlsm файл в Excel
# 3. Alt + F11 → Insert → Module
# 4. Скопируйте код из excel_macro.vba
# 5. Сохраните
```

Теперь можно запускать через **Tools → Macro → Macros → LoadNSPDData**
